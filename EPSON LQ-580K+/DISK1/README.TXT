以下包含EPSON Status Monitor 3的最新信息。
                                                          09/10/99
---------------------------------------------------------------------
1.一般问题解决方式

1-1. 使用EPSON Status Monitor 2 和EPSON Status Monitor 3的注解

     EPSON Status Monitor3 监视以下类型的连接，尽管这些设置对于EPSON 
     Status Monitor 2是无效的。
      * 本地打印机
      * LPR 打印机
      * 网络打印机
      * Windows 共享打印机
     
     这些设置在大多数场合下应该起作用。

      如果你安装了EPSON Status Monitor 2并更改了监视设置，它也将改变
      EPSON Status Monitor 3的设置。这将引起打印和监视上的问题。
     (注)   *如果特殊的通信协议未安装的话，监视设置将无效。
            *设置EPSON Status Monitor 3, 请重新安装你的打印机驱动程序。 

1-2. 如果一个共享打印机不能被监控的话，检查是否进行了下列设置：

      * 确认打印机驱动程序已被安装到可共享打印机的计算机上，该打印机被
        设为共享，并且在打印机驱动程序的“监视属性”中选择了“允许监视
        共享打印机”。
   
      * (只有Windows98/95) 打开正共享打印机的计算机上控制面版的“网络”
        图标, 检查已安装Microsoft 网络的文件和打印机共享控件。

      * (只有Windows98/95)检查IPX/SPX兼容协议既未安装到客户端，也未安装
        到服务器端。


2. Windows NT 环境的解决方法

2-1. 当从多用户Windows NT环境卸载EPSON Status Monitor 3时，卸载前关闭所
     有客户端的快捷图标。你可以在监视属性中关闭快捷图标。
2-2. 在未升级的Windows NT环境下使用Windows NT 4.0共享打印机时，如果你选
     择了“允许监视共享打印机”，然后启动EPSON Status Monitor 3,出现应用
     错误通知窗口并且你无法监视网络共享打印机。为避免这个问题发生，确认你
     已安装了Windows NT 4.0 Service Pack 2 或更高版本。
2-3. 当使用Windows NT/2000 共享打印机时, 如果你选择了“允许监视共享打印机”，
     然后启动EPSON Status Monitor 3, 你不能监视网络共享打印机。为避免这个
     问题发生，打开控制面版的“服务”图标，停止EPSON 打印机状态代理服务，
     并启动EPSON Status Monitor 3。启动EPSON Status Monitor 3后, 重新启动
     打印机状态服务。你每次登录Windows NT/2000 时都必须这么做。
     

3. 网络打印机的解决方法

3-1. 为了监视网络打印机，每个打印队列中只要连接一个打印服务。参看你的用户指南。

3-2. 通过IPX路由器你不能够监视网络队列打印机是因为IPX 跳步的设置为0。

3-3. 当监视NDS打印机时，必须使用Novell提供的客户服务:

     当使用Windows NT: Novell IntranetWare Client for Windows NT
     当使用Windows 95: Novell IntranetWare Client 32 for Windows 95

3-4. 监视NDPS 打印机

    
    该产品支持象Bindery或NDS服务器这种基于队列打印系统所配置的网络打印机。
    你不能够监视NDPS (Novell Distributed Printing Services) 打印机。


4. 针式打印机使用 EPSON Status Monitor 3 的解决方法

4-1. 打印时错误和警告提示注解

    *如果连接到一台网络打印机并且在打印时对其进行监视，确认正在运行EPSON Status Monitor 3 。

    *如果连接到一台LPR 打印机并且在打印时对其进行监视，确认正在运行EPSON Status Monitor 3 。

    *当从Windows 98/95连接到Windows NT 共享打印机时, 在打印期间，是不可能显示
     错误和警告提示的。确认正在运行EPSON Status Monitor 3, 并使用打印机状态监
     视窗口查找错误。

    *当通过路由器连接一台共享打印机时，在某些情形下，由于路由器的设置，在打
     印期间共享打印机的错误和警告提示是不可用的。
